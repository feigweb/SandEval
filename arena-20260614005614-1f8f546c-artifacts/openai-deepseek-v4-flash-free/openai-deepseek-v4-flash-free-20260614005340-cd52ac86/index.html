Google

Google 是一家美国跨国科技公司，专注于搜索引擎、云计算、广告技术以及人工智能等领域。Google 由拉里·佩奇和谢尔盖·布林在斯坦福大学攻读博士期间创立，目前是 Alphabet Inc. 的全资子公司。

历史

1996年，拉里·佩奇和谢尔盖·布林在斯坦福大学开始研究"BackRub"项目——一个使用链接分析来确定网页重要性的搜索引擎。1997年他们注册了 google.com 域名。1998年9月4日，Google 公司在加州门洛帕克的一个车库中正式成立。

2004年，Google 在纳斯达克上市（股票代码 GOOGL）。此后，公司通过收购和内部研发不断扩展业务，包括收购 YouTube（2006）、推出 Android（2007）、Chrome（2008）等。2015年，公司重组为 Alphabet Inc.，Google 成为其最大的子公司。

近年来，Google 大力投入人工智能，推出了 Google Assistant、TensorFlow 开源框架、Bard（后更名为 Gemini）等产品，并在 2023–2024 年加速 AI 在搜索和云服务中的融合。

主要产品

  • Google 搜索 — 全球最受欢迎的搜索引擎，市场份额超过 90%。
  • Android — 基于 Linux 的移动操作系统，全球装机量超过 30 亿。
  • YouTube — 全球最大的视频分享平台，月活跃用户超过 25 亿。
  • Google Cloud — 云计算服务，包括 GCP、Workspace 等。
  • Chrome — 最流行的 web 浏览器。
  • Gemini — 多模态 AI 模型系列（前身为 Bard）。

公司文化

Google 以其独特的公司文化而闻名，包括著名的"20% 时间"政策（工程师可将 20% 工作时间用于个人项目）、免费餐饮、健身房以及开放办公环境。公司使命是"整合全球信息，使人人皆可访问并从中受益"。

Microsoft（微软）是一家美国跨国科技公司，由比尔·盖茨和保罗·艾伦于 1975 年创立。它是全球最大的软件公司之一，以 Windows 操作系统、Office 办公套件和 Azure 云服务闻名。

历史

1975年，比尔·盖茨和保罗·艾伦在新墨西哥州阿尔伯克基创立了 Microsoft，为 Altair 8800 开发 BASIC 解释器。1985 年发布了 Windows 1.0，开启了图形用户界面的新时代。1990 年代，Windows 和 Office 成为行业标准，Microsoft 迅速成长为科技巨头。

2000 年代，公司面临反垄断诉讼，但也在 Xbox 游戏主机、企业软件等领域持续扩展。2014 年萨提亚·纳德拉出任 CEO，提出"移动为先、云为先"战略，大力推动 Azure 云服务和跨平台软件。

近年来，Microsoft 在 AI 领域投入巨大，投资 OpenAI，将 GPT 模型整合到 Bing、Office、Azure 等产品中，并推出 Copilot 系列 AI 助手。

主要产品

  • Windows — 全球最广泛使用的桌面操作系统。
  • Microsoft 365 — 包括 Word、Excel、PowerPoint 等办公套件。
  • Azure — 全球第二大云计算平台。
  • Xbox — 游戏主机及 Game Pass 订阅服务。
  • GitHub — 全球最大的代码托管平台（2018 年收购）。
  • Copilot — 基于 GPT-4 的 AI 助手。

公司文化

Microsoft 早期的文化以"精英管理"和"激烈竞争"著称。在纳德拉领导下，公司文化转向"成长型思维"（growth mindset），强调协作、多样性和持续学习。Microsoft 的使命是"赋能地球上的每一个人和每一个组织，成就更多"。

Apple Inc.（苹果公司）是一家美国跨国科技公司，以 iPhone、iPad、Mac 等消费电子产品以及 iOS、macOS 等软件平台闻名。苹果是全球市值最高的公司之一，以其创新设计和封闭生态系统著称。

历史

1976 年 4 月 1 日，史蒂夫·乔布斯、史蒂夫·沃兹尼亚克和罗恩·韦恩在加州创立了苹果电脑公司。Apple I 和 Apple II 开启了个人电脑革命。1984 年推出的 Macintosh 以其图形界面和鼠标操作震惊世界。

1985 年乔布斯被迫离开苹果，公司一度陷入困境。1997 年乔布斯回归后，推出了 iMac、iPod、iPhone（2007）和 iPad（2010）等革命性产品，使苹果成为全球最有价值的公司之一。2011 年蒂姆·库克接任 CEO，继续推动产品线扩展和服务业务增长。

主要产品

  • iPhone — 重新定义了智能手机，是苹果最畅销的产品。
  • Mac — 包括 MacBook Air/Pro、iMac、Mac mini、Mac Pro 等。
  • iPad — 平板电脑市场的标杆产品。
  • Apple Watch — 全球最受欢迎的智能手表。
  • AirPods — 无线耳机，引领了 TWS 市场。
  • Apple 服务 — App Store、Apple Music、iCloud、Apple TV+ 等。
  • Apple Silicon — M1/M2/M3/M4 自研芯片。

公司文化

苹果以极简主义设计、注重细节和保密文化闻名。乔布斯时代的"现实扭曲场"和追求完美的精神深深烙印在公司 DNA 中。苹果的使命是通过创新产品丰富人们的生活，同时坚持对用户隐私和环保的承诺。

OpenAI

网站openai.com

OpenAI 是一家美国人工智能研究公司，致力于构建安全且有益的通用人工智能（AGI）。其开发的 GPT 系列大语言模型和 DALL·E 图像生成模型引领了生成式 AI 的浪潮，ChatGPT 已成为史上增长最快的消费者应用之一。

历史

2015 年 12 月，萨姆·奥尔特曼、埃隆·马斯克、伊利亚·苏茨克弗等人在旧金山共同创立了 OpenAI，最初是一家非营利人工智能研究机构。2018 年马斯克离开董事会。2019 年，OpenAI 转型为"有上限利润"的公益公司（OpenAI LP），并接受了微软 10 亿美元的投资。

2020 年发布 GPT-3（1750 亿参数），震惊业界。2022 年 11 月推出 ChatGPT，两个月内用户突破 1 亿。2023 年 3 月发布 GPT-4，同年推出 GPT-4 Turbo、DALL·E 3 和语音功能。2024–2025 年发布了 GPT-4o、o1 推理模型、Sora 视频生成模型等。

公司治理方面，2023 年 11 月经历了一次戏剧性的董事会解雇和重新聘任 CEO 事件，凸显了 AI 安全治理的复杂性。

主要产品

  • ChatGPT — AI 聊天助手，支持文本、图像、语音多模态交互。
  • GPT-4 / GPT-4o — 多模态大语言模型，通过 API 提供服务。
  • DALL·E 3 — 文本到图像生成模型。
  • Whisper — 开源语音识别系统。
  • OpenAI API — 开发者平台，提供 GPT、DALL·E 等模型接口。
  • Sora — 文本到视频生成模型（2024 年发布）。
  • o1 / o3 — 推理增强模型系列。

公司文化

OpenAI 的文化根植于"安全构建 AGI"的使命。公司强调长线思维、跨学科协作和透明度（早期），但随着商业化推进，保密性有所增加。OpenAI 的治理结构独特——由非营利母公司控制营利子公司，以确保 AGI 造福全人类。公司核心原则包括：广泛受益、安全导向、技术领先和合作精神。
